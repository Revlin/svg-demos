Raphael JS Vector Graphics
==========================

In order to really appreciate and relate to the code examples that you encounter
when reading the book, you are *strongly* encouraged to refer to these demos.

Using the Source Code
---------------------

In order to utilise this source code, you should open the index.html in your browser
- this file provides links to every example contained within subdirectories and is
the most convenient way of seeing the examples in action as you work through the book.
